# 🎓 EduLembra
### Plataforma de Revisão Inteligente para Estudantes

O **EduLembra** é uma plataforma educacional criada para ajudar estudantes — especialmente do ensino público — a revisar conteúdos de forma eficiente, utilizando princípios de **revisão espaçada**, organização de tarefas e acompanhamento de progresso.

O projeto foi desenvolvido como um **UX/UI Case Study**, com foco em acessibilidade, simplicidade e impacto social.

---

## 🚀 Tecnologias Utilizadas
- **HTML5**
- **CSS3 (Flexbox, Grid, Responsividade, Animações)**
- **JavaScript**

---

## 🎯 Objetivos do Projeto
- Ajudar estudantes a **não esquecer conteúdos importantes**
- Criar uma interface simples, acessível e intuitiva
- Estimular autonomia e organização nos estudos
- Demonstrar um processo completo de **UX/UI Design**
- Construir um projeto de impacto educacional para portfólio

---

## ✨ Principais Funcionalidades
- 📚 **Agenda de estudos com lembretes**
- 🔁 **Sistema de revisão espaçada**
- 📊 **Acompanhamento de progresso**
- 🧠 **Interface focada em clareza e foco**
- 📱 **Totalmente responsivo (mobile, tablet e desktop)**
- 🎨 **UI leve, moderna e acessível**

---

## 🧠 Processo de UX/UI
- Pesquisa e entendimento do problema
- Definição de personas
- Mapeamento de jornada do usuário
- Arquitetura da informação
- Wireframes
- Interface visual (UI)
- Protótipo navegável
